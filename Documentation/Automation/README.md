This folder stores project-local ledgers and reports for the Sc2 bot documentation deepening automations.

## Usage Notes

- `Sc2BotTerranDocumentationAutomationReport.md` tracks the current open documentation gaps or ambiguities for owned Terran code.
- `Sc2BotApiRefinementLedger.md` is the topic source for the API and ecosystem deepening loop.
- `Sc2BotApiSkillAutomationReport.md` tracks the current open findings or ambiguities that remain after the most recent API-skill refinement pass.
- `Sc2BotGoalActionCoverageMatrix.md` is the current owned-Terran goal, action, and offline sanity coverage matrix.
- `Sc2BotGoalActionCoverageAutomationReport.md` tracks the current unresolved goal or action coverage gaps after the latest offline coverage run.
