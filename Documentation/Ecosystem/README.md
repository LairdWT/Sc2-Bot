# Ecosystem Leverage Notes

## Purpose

This folder records external patterns that may inform `L:\Sc2_Bot`, after local source review is complete.

These notes are leverage-oriented, not authoritative. They exist to answer questions such as:

- what other C++ and Python bot developers have already solved
- which deterministic placement or scheduling patterns can be adapted cleanly
- which ideas conflict with the current Terran architecture and should remain external only

## Guardrails

- Local source remains the authority for current behavior.
- External bot patterns should only be documented here when they directly inform our design goals.
- Do not copy external patterns into this repository documentation without first stating the local integration seam they would affect.
